Instrucciones del Juego

Inicio del juego:

  • Abre el archivo INDEX.HTML del juego en tu navegador web.
  • Asegúrate de que el juego esté cargado correctamente y visible en la ventana del navegador.

Controles del juego:

  • Correr: el personaje corre automáticamente al inicio del juego.
  • Saltar: Presiona la barra espaciadora para hacer que el personaje realice un salto.
  • Durante el salto, el personaje intentará evitar los obstáculos para tener un salto exitoso y sumar puntos.
  • Caer: El personaje caerá automáticamente después de completar un salto o si no se realizó ningún salto.

Objetivos y puntuación:

  • El objetivo principal del juego es correr y evitar obstáculos para obtener la mayor puntuación posible.
  • Cada vez que el personaje realice un salto exitoso sobre un obstáculo (tronco o piedra), se sumarán 10 puntos a la puntuación total.
  • Si el personaje choca con un obstáculo, se restarán 10 puntos de la puntuación total.

Colisión con obstáculos:

  • Si el personaje choca con un tronco o una piedra, se reproducirá un sonido y se mostrará una imagen intermitente con el personaje golpeado.
  • Además, se restarán 10 puntos de la puntuación total.
  • El personaje puede levantarse rapidamente del golpe presionando la barra espaciadora para activar nuevamente el salto.

Obtención de tiempo:

  • Si el personaje agarra una lámpara de energía, se reproducirá un sonido de energía.
  • Al obtener una lámpara de energía se le sumarán 5 segundos al tiempo de juego.

Finalización del juego:

El juego finaliza cuando se agota el tiempo. Se mostrará un mensaje con el puntaje obtenido y la opción de retornar a la pantalla Principal.
